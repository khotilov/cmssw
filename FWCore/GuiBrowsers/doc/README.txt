------------------------------------------------------------------------
HOW TO RUN CONFIGBROWSER/CONFIGEDITOR

Enter your CMSSW environment (any >=CMSSW_2_1_X):
   cmsrel CMSSW_3_3_4
   cd CMSSW_3_3_4/src
   cmsenv
And run:
   edmConfigEditor

------------------------------------------------------------------------
HOW TO RUN EDMBROWSER

Enter your CMSSW environment (any >=CMSSW_3_3_X):
   cmsrel CMSSW_3_3_4
   cd CMSSW_3_3_4/src
   cmsenv
And run:
   edmBrowser
