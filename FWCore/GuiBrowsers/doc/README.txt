HOW TO RUN CONFIGBROWSER

Enter your CMSSW environment (any >=CMSSW_2_1_0):
   cmsrel CMSSW_2_1_17
   cd CMSSW_2_1_17/src
   cmsenv
And run:
   edmConfigBrowser
